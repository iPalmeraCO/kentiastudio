Cookie Notice is a simple plugin which shows notice to your visitors which tells your site uses EU law for Cookies.

== Description ==

[Cookie Notice](http://analyticator.com/wordpress/plugins/add-cookie-notice/) Compliance is a EU law and it is important for site owners to show a notice to visitors to accept the cookies.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/add-cookie-notice` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Cookie Notice screen to configure the plugin
1. Basic Settings and Advanced Settings for more features.
